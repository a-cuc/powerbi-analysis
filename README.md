# Examining the Labor Market with Power BI
A project combining job-posting data and labor statistics for a report
- Labor statistics extracted from the [PSA Labor Force Survey](https://psa.gov.ph/statistics/labor-force-survey)
- Job posting data sourced from [Jooble](https://jooble.org/api/about) and [Careerjet](https://www.careerjet.com/partners/api/) API
